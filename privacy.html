PRIVACY POLICY

Sight Line collects no personal and sensitive user data (it collects no user data whatsoever) and does not use any user data or share any user data with any other parties. The app simply retrieves the information about a works site from our servers after a beacon is detected and displays it to the user. No record is collected of which app installations requested the information for which beacons.

As no user data is collected there can be no procedure to handle it and there can be no process for retention and deletion as there is no user data to handle retain or delete.
